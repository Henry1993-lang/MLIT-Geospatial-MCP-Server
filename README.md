# GeoAppraisal AI (不動産鑑定士向けAI調査支援アプリ)

このアプリは、不動産鑑定士の方々が地理空間情報（地図データなど）を簡単に調査できるように作られた、AIアシスタント機能付きのWebアプリです。

国土交通省の「不動産情報ライブラリ」からデータを取得し、地図・チャット・表を組み合わせて、分かりやすく情報を表示します。プログラミングの知識がなくても、以下の手順に沿って準備をすればご自身のパソコンで動かすことができます。

---

## 📂 アプリの構成

このアプリは3つの部分から成り立っています：
1. **フロントエンド (UI)**: ブラウザで表示される画面（React + Vite）
2. **バックエンド (API)**: データ処理の裏側（Express + Node.js）
3. **MCPサーバー**: 国土交通省のデータを実際に取得するプログラム（Python）

---

## 🔑 APIキーの取得方法（任意）

実際のデータを使ってアプリを動かすためには、「APIキー」と呼ばれる専用のパスワードのようなものが必要です。
※取得しなくても、お試し用の「モックデータ（ダミーデータ）」で画面の動きを確認することは可能です。

### 1. Gemini APIキー (AIチャット用)
AIとチャットでやり取りをするために、GoogleのAI「Gemini」のキーを取得します。（無料で取得できます）

1. [Google AI Studio](https://aistudio.google.com/app/apikey) にアクセスします。
2. Googleアカウントでログインします。
3. 画面内の「Create API key」ボタンをクリックします。
4. 表示された文字列（`AIzaSy...` のような文字列）をコピーして、どこかにメモしておきます。

### 2. 不動産情報ライブラリ APIキー (国土交通省のデータ取得用)
地図上に表示する実際の不動産データを取得するために取得します。（無料で取得できます）

1. [不動産情報ライブラリ API提供について](https://www.reinfolib.mlit.go.jp/api/) にアクセスします。
2. ページ内の「利用者登録」からアカウントを作成し、ログインします。
3. ログイン後のマイページから「APIトークン発行」等を選択します。
4. 利用規約に同意し、表示されたAPIキー（トークン）の文字列をメモしておきます。

取得したこれら2つのキーは、後ほど「**ステップ4**」でアプリに設定します。

## 🚀 導入手順（セットアップ）

ターミナル（Macの場合は「ターミナル」、Windowsの場合は「コマンドプロンプト」や「PowerShell」）を開いて、順番にコマンド（命令）をコピー＆ペーストして実行してください。

### ステップ1: このアプリをダウンロードする

まず、任意のフォルダでこのアプリのデータをダウンロード（clone）します。

```bash
# アプリのデータをダウンロード
git clone https://github.com/Henry1993-lang/MLIT-Geospatial-MCP-Server.git

# ダウンロードしたフォルダの中に移動
cd MLIT-Geospatial-MCP-Server
```
*(※すでにダウンロード済みの場合は `git pull` で最新版に更新できます)*

### ステップ2: MCPサーバー（データ取得プログラム）の準備

アプリのフォルダ（`MLIT-Geospatial-MCP-Server`）の中で、別のデータ取得用プログラムをダウンロードして設定します。

```bash
# MCPサーバーをダウンロード
git clone https://github.com/chirikuuka/mlit-geospatial-mcp.git

# MCPサーバーのフォルダに移動
cd mlit-geospatial-mcp

# Pythonの専用環境（仮想環境）を作成
python3 -m venv .venv

# 仮想環境を起動（Mac/Linuxの場合）
source .venv/bin/activate
# ※Windowsの場合は `.\.venv\Scripts\activate`

# 必要なプログラムをインストール
pip install -e .

# 元のフォルダに戻る
cd ..
```

### ステップ3: アプリ本体の準備

画面（フロントエンド）と裏側（バックエンド）を動かすための準備をします。

```bash
# フロントエンドの準備
cd app/frontend
npm install

# バックエンドの準備
cd ../backend
npm install

# 設定ファイル（.env）のひな形をコピーして作成
cp .env.example .env
```

### ステップ4: APIキーの設定（任意）

`app/backend` フォルダの中に `.env` というファイルが作成されています。このファイルをテキストエディタで開き、APIキーを書き込みます。

```env
# 取得したGeminiのAPIキーを入力してください
GEMINI_API_KEY=ここにGemini_APIキーを書く

# 取得した不動産情報ライブラリのAPIキーを入力してください
LIBRARY_API_KEY=ここに不動産情報ライブラリ_APIキーを書く

# 下記2行はそのままでOKです
MCP_SERVER_COMMAND=../../mlit-geospatial-mcp/.venv/bin/python
MCP_SERVER_ARGS=../../mlit-geospatial-mcp/src/server.py
```
*(※APIキーを持っていなくても、未設定のまま起動すればお試しモードで動きます。)*

## ▶️ アプリの起動方法

アプリを動かすには、「バックエンド」と「フロントエンド」の両方を起動する必要があります。ターミナルの画面を2つ（タブを2つ）開いて実行します。

### 1つ目のターミナル（バックエンド用）
```bash
cd app/backend
npm run dev
```

### 2つ目のターミナル（フロントエンド用）
```bash
cd app/frontend
npm run dev
```

両方が起動したら、お使いのブラウザ（ChromeやSafariなど）で以下のURLを開いてください。
👉 **http://localhost:5173**

## 📖 使い方

1. 画面左側に**地図**が表示されるので、調査したいエリアを表示します。
2. 画面右側の**チャット**に、調べたい内容を自然な日本語で入力します。
3. AIが内容を理解し、自動的にデータを取得してくれます。
4. 取得した結果は、地図上の**ピン**や画面下部の**データテーブル（表）**に表示されます。

**【チャットの入力例】**
- 「東京駅周辺の取引事例を教えて」
- 「対象地から1km以内の住宅地取引を抽出して」
- 「この周辺の地価公示データを確認して」


## 💡 補足と注意事項

- `.env` ファイル（パスワードのようなもの）は自分のパソコンの中にだけ保存されます。インターネット上には公開されないよう設定されているので安心してください。
- APIキーが正しく設定されていない、またはキーを持っていない場合は、安全のため「モックデータ（ダミーデータ）」が表示されるお試しモードになります。
